# VisionVault Changelog v1.1.0

## Major Update — Web UI Rewrite & Platform Expansion

This update introduces a full overhaul of the VisionVault Web UI, transforming it into a more complete, TV-style media frontend with improved layout structure, playback flow, and platform consistency.

---

## New Features

### Rebuilt VisionVault Web UI

* Fully redesigned the VisionVault web interface from the ground up
* Introduced a more modern, TV-style layout with cleaner structure and improved navigation
* Improved overall responsiveness and visual consistency across screen sizes and devices
* Established a stronger foundation for future feature expansion

---

### Hero Section & Featured Content

* Added a dynamic **hero section** on the home page
* Displays featured or recently interacted content prominently on launch
* Improved first-impression experience for TV and browser users

---

### Improved Content Rows & Layout

* Reworked horizontal browsing rows for smoother navigation and better spacing
* Improved poster alignment and scaling for a cleaner presentation
* Reduced UI clutter and overlapping elements across all sections

---

### Dedicated Library Pages

* Added a full **All Movies** page for complete library browsing
* Improved separation between home content and full library views
* Better navigation flow between featured content and full collections

---

### Enhanced Poster & Artwork Handling

* Improved support for animated posters in the web UI
* Refined poster prioritization between static and animated artwork
* Cleaner poster presentation with fewer overlays and distractions

---

### Web Player Improvements

* Improved browser-based playback stability and performance
* Enhanced subtitle detection and loading behavior
* Better support for external subtitle files and common folder structures
* Improved resume playback handling

---

### 3D Playback Preparation (Foundation)

* Began groundwork for future **3D viewing modes** in the web UI (SBS, etc.)
* Updated player structure to support advanced rendering features in future updates

---

## Improvements

### TV Experience Refinement

* Improved couch-friendly layout for large-screen viewing
* Better scaling and spacing for posters, rows, and hero elements
* Reduced visual noise for a more immersive browsing experience

---

### Navigation & Interaction

* Improved focus behavior and selection flow across the UI
* Cleaner transitions between pages and content views
* More intuitive browsing experience overall

---

### Shared Backend Integration

* Strengthened connection between desktop/server and web UI
* Improved data flow for movie details, playback state, and library structure
* Continued aligning VisionVault toward a unified multi-platform system

---

## Fixes

### Layout & Rendering Issues

* Fixed overlapping elements in certain UI sections
* Resolved scaling issues that caused stretched or misaligned layouts
* Improved consistency across different screen sizes and resolutions

---

### Poster Display Issues

* Fixed cases where posters would render incorrectly or with unwanted overlays
* Improved handling of missing or mismatched artwork

---

### Playback & Subtitle Handling

* Fixed subtitle detection inconsistencies in some edge cases
* Improved reliability of resume playback behavior

---

## Notes

* This update lays the groundwork for the upcoming **VisionVault Android TV app**
* The Web UI is now positioned as a full-featured frontend, not just a companion interface
* Future updates will continue expanding playback features, 3D support, and cross-device syncing
